## Gomoku AI
> 오목은 한 줄로 5개 이상의 돌을 먼저 놓는 사람이 승리하는 게임입니다. 본 프로젝트에서는 Minimax 알고리즘을 기반하여 오목 AI를 제작하였습니다.
알고리즘은 Swift로 작성되었으며, iOS 앱을 통해 실행 가능합니다.
